Favourites Upload Utility                                                 ===============================================================================================Date installed: 18/06/2008Author: Matt Barfoot - Clearview Webmedia LimitedContact: matt.barfoot@clearview-webmedia.co.uk / 01727 838352 / 07968 175 795Description===============================================================================================Nightly (12:00 AM) extract which populates a Favourites DB from several extracts from the Sage 200 Live  database. After all extracts are completed an WDDX XML file is generated  and sent to the vegetarianexpress.co.uk website via FTP. This is later import by the website.Install Location===============================================================================================Hostname: dbserver (dbserver.vegexp.local)Configuration===============================================================================================Coldfusion service (NT Service - "Coldfusion 8 Application Server")  ColdFusion script: getFavourites_Sage200.cfm in e:\ColdFusion8\wwwrootComponents===============================================================================================Coldfusion 8.01SQLServer database - VESAGEFAVOURITESDependencies===============================================================================================Local TCP Port 8500 - ColdFusion ServiceLocal TCP Port 2930 - JNDINAT/Firewall configured to route/allow FTP between 10.0.0.4 and 80.79.137.16SQLServer login/user  username: sagews password sagewsdb_dataread on VEGEXPLIVEdb_owner on VESAGEFAVOURITES
